Snake Game

The Snake Game is a classic arcade game implemented using HTML, CSS, and JavaScript. The objective is to control a snake to eat food, growing longer with each meal, while avoiding collisions with the walls and the snake's own body. The game keeps track of your score and saves the high score in local storage, providing a fun and challenging experience. The addition of a pause functionality allows players to pause and resume the game seamlessly, enhancing the overall gameplay.

Electricity Bill Management System

The Electricity Bill Generator is a desktop application developed using Java and Java Swing within the Eclipse IDE. This application allows users to input their electricity consumption details and automatically calculates the total bill based on predefined tariff rates. Featuring a user-friendly graphical interface, it streamlines the billing process, ensuring accuracy and efficiency. The application can handle various customer data, generate detailed bill summaries, and provide print options, making it a practical tool for utility companies and consumers alike.
